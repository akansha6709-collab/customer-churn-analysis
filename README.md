# 📊 Customer Churn Analysis

## 📌 Objective
The goal of this project is to analyze customer churn behavior and identify key factors contributing to customer attrition in a telecom company.

---

## 🛠️ Tools & Technologies
- SQL (Data Analysis)
- Power BI (Data Visualization & Dashboard)

---

## 📁 Dataset
Telecom customer dataset containing:
- Customer demographics
- Services subscribed
- Payment and billing information
- Churn status

---

## 🔍 Key Insights
- Customers on **Month-to-Month contracts** have the highest churn rate.
- **New customers (0–6 months)** are more likely to churn.
- Customers with **higher monthly charges** show higher churn behavior.
- **Electronic check payment method** has higher churn compared to other methods.

---

## 📊 Dashboard Features
- Total Customers, Churn Rate, and Churn Count KPIs
- Churn distribution by Contract Type
- Churn analysis by Tenure Band
- Monthly charges comparison (Churn vs Retained)
- Customer retention overview

---

## 💡 Business Recommendations
- Promote long-term contracts with incentives
- Focus on retaining new customers through onboarding strategies
- Improve service quality for high-paying customers
- Encourage automatic payment methods to reduce churn risk
